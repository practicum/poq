330 Taylor 850

dept low_wage
101 2
209 0

Query B

dept low_wage
101 2
330 1
Query A:
SELECT dept, sum( case when salary < 1000 then 1 else 0 end ) as low_wage FROM Employee GROUP BY dept;
Query B:
SELECT dept, count(1) as low_wage FROM Employee            WHERE salary < 1000 GROUP BY dept;
